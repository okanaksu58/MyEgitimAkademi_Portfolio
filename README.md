# MY Yazılım - Eğitim Akademi ve Danışmanlık Projesi

Bu proje, MY Yazılım çatısı altında Eğitim Akademi ve Danışmanlık hizmetleri kapsamında geliştirilen bir web uygulamasıdır. Projede kullanıcıların eğitim içeriklerine, danışmanlık hizmetlerine ve akademik duyurulara ulaşabileceği bir yapı tasarlanmıştır.

## Proje Özellikleri
- Eğitim kurslarının listelenmesi ve detay sayfaları
- Danışmanlık hizmetlerine başvuru formu
- Akademik etkinlikler ve haberler bölümü
- Admin paneli üzerinden içerik yönetimi
- Responsive (mobil uyumlu) tasarım

## Kullanılan Teknolojiler
- ASP.NET Core MVC
- Entity Framework Core
- Microsoft SQL Server
- Bootstrap
- Katmanlı Mimari (Business, DataAccess, Entity, Presentation)
